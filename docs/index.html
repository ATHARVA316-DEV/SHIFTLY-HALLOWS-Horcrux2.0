Horcrux Assistive App — Demo

This repository contains a Streamlit frontend and a Flask backend (camera/ONNX). GitHub Pages can only host static files, so this page serves as a documentation/demo landing page.

How to view/run the real app

  • Run locally with Docker (recommended). See deploy/RENDER.md and the deploy README.
  • Or run the Flask + Streamlit apps locally (requires Python, OpenCV, ONNX):
docker build -t shiftly-horcrux:latest .
docker run -p 8501:8501 -p 9999:9999 shiftly-horcrux:latest

What works on GitHub Pages

This static site can include screenshots, usage instructions, and links to a hosted deployment (Render/Fly/Railway). Interactive camera/ML features require a server and cannot run on GitHub Pages.
